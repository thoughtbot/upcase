Design for Developers

Resources & Reading List

Design Fundamentals

  • Design Principles at Learn

    Directed Visual Principles learning tools with validation steps

  • The Visual Display of Quantitative Information

    The quintessential information design book.

  • A Practical Guide to Designing for the Web

    Great book for learning some fundamental design principles and how they apply to the web

  • Graphic Design: The New Basics

Grids

  • Grid Systems: Principles of Organizing Type

    If you only read one book on grid systems this is the one. It does an amazing job of explaining and using grid systems.

  • Modulator Grid Pattern

    Helps generate custom grid patterns to throw in th background of your site while building.

  • Making and Breaking the Grid: A Graphic Design Layout Workshop
  • Gridulator

    Grid calculator for columns and gutter

  • Skeleton CSS Grid

    Adaptive 16 column grid framework and base styles.

  • 960 grid system

    The traditional fixed web grid. Comes in 12 columns and 16.

  • Grid overlay bookmarklet

    Overlays grid on top of your site. Helpful when building out your grid on the web.

Typography

  • Typography on Learn

    Directed Typography learning tools with validation steps

  • Modular Scale

    Calculator to help build a modular scale within your type.

  • The Elements of Typographic Style Applied to the Web

    The greatest book on type as applied to websites.

  • Jason Santa Maria - On Web Typography

    Video about exicuting typography on the web.

  • Detail In Typography
  • I Love Typography

    Blog for all things in the type world.

  • Revised Font Stack

    Reference list for good font stacks.

  • Typekit

    One of the few sites to find high quality typeface that can be embedded on the web.

  • fontsquirrel

    Downloadable fonts that have varying quality and licences to add to your site.

  • Google web fonts

    Littered with some terrible typeface designs but there are some diamonds in the rough.

Color & Texture

  • Color: a color matching game

    Game that helps you match color and get good at color pairing.

  • My Secret for Color Schemes

    Trick in Photoshop to help color matching.

  • Colors App

    Mac color picker app.

  • Subtle patterns

    Slight patterns for background noise.

  • Sass Color Functions

    Article about color about how to purposefully use color functions in Sass.

Icons

  • Icons Guide
  • NounProject

    Open source icon project.

  • Icon Sweets

    Free minimal icons for Photoshop.

  • Icon Monstr
  • The Toolbox

    A bunch of design resources.

  • Symbolset

HTML, CSS & Sass

  • CSS on Learn

    Directed CSS learning tools with validation steps

  • Sass on Learn

    Directed Sass learning tools with validation steps

  • HTML5 for web designers

    This book covers all of the changes in HTML5. A quick succinct read.

  • Bourbon

    thoughtbot’s open source Sass library

  • CSS3 for web designers

    Another book from A Book Apart. It covers all the new CSS3 styles and selectors.

  • Dive Into HTML5

    A plain language explanation and reference for all of the HTML5 elements.

  • Responsive Web Design

    THE book on responsive design. At this point it’s a must read for any designer on the web.

  • CSS3: A Practical Introduction

    Presentation for CSS3

  • The 30 CSS Selectors you Must Memorize

    Simply a list of CSS selectors that are very helpful.

  • When can I use...

    Reference for which browsers support which CSS & HTML features.

  • Normalize

    The default reset right now for CSS developers.

  • Animate CSS

    Good library for simple CSS animations.

  • Dan Cederholm - Handcrafted CSS

    Good video to watch about crafting your CSS.

UX & Design thinking

  • Don’t Make Me Think

    The classic book on usability on the web.

  • A List Apart

    A periodical for web designers. A must read every month.

  • Wireframing for web apps

    Article on wireframing and sketching for web apps

  • What happens to user experience in a minimum viable product?
  • Design Staff

    Google Ventures design blog.

  • UIE Virtual Seminars

    Seminars from the premiere usability experts on current web design topics

  • Designing With Emotion
  • Creative Morning’s Talks

    Collection of talks from Creative Mornings, a free lecture series around the world. They are great for motivation, inspiration and deign thinking.

  • Wilson Miner - When We Build

    Video about our thinking as designers and what we do when we put things into the world.

  • Frank Chimero - The Shape of Design

    A gallery of websites.

  • ui parade

    UI design inspiration

  • ffffound

    An assortment of design and art related images. Not always SFW.

  • Dribbble

    A design community of 400x300 pixel shots.

  • Patterntap

  • Gimmebar & Zootool

    Visual bookmarking tools that can be used as an inspiration collection tool.

  • Littlesnapper

    A Mac app that collects all types of inspiration and helps you categorize them.
